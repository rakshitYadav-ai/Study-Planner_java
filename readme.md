# 📚 Study Planner (Java Project)

## 🔍 Overview

This project is a command-line based Study Planner developed in Java. It helps students organize their study schedule by distributing topics across available days and tracking their progress.

The goal of this project is to provide a simple yet effective way for students to manage their preparation during exams.

---

## 🎯 Problem Statement

Students often face difficulty in managing multiple subjects during exams. Without proper planning, they tend to either over-focus on certain subjects or neglect others.

This project addresses that problem by:

* Structuring study plans
* Distributing workload evenly
* Tracking daily progress

---

## 🚀 Features

* Add multiple subjects with exam timelines
* Automatically generate a study schedule
* View day-wise topic distribution
* Mark tasks as completed
* Track overall progress for each subject

## ▶️ How to Run

1. Clone or download the project

2. Open terminal in project folder

3. Compile all files:

   javac *.java

4. Run the program:

   java Main
